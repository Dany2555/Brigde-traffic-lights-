This project is a simple traffic light system built using an Arduino, three LEDs (red, yellow, and green), resistors, a push button, and a breadboard. The system works by controlling the LEDs to simulate real traffic lights. The green LED turns on first, allowing movement, followed by the yellow LED as a warning, and then the red LED to stop traffic. A push button is included to control the system. When pressed, it can start or stop the traffic light sequence. The project helps in understanding basic electronics, circuit connections, and programming using Arduino.
