# 🚦 Bridge Traffic Lights Project

## 📌 Description
This project simulates a traffic light system using an Arduino, three LEDs (red, yellow, green), and a push button. The LEDs represent stop, wait, and go signals, just like a real traffic light.

## ⚙️ Components Used
- Arduino board  
- Breadboard  
- 3 LEDs (Red, Yellow, Green)  
- 4 Resistors  
- Push Button  
- Jumper wires  

## 🔌 Circuit Explanation
Each LED is connected to a digital pin on the Arduino through a resistor.  
The push button is connected to another pin and is used to control the system (ON/OFF).
